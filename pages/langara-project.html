Opportunity

Langara-app.ca is the website for the Web and Mobile App Design and Development program (WMDD) at Langara. Although it has been live for a couple of years, the program and the college seemed to have 'abandoned' it. I was selected among all the students and hired, along with two developers, to update it.

Project Goals

  • Revise and update the existing user flow.
  • Update the UI kit in alignment with new pages and components.
  • Preserve information and the main structure while introducing a fresh, implementable look.
  • Update the homepage, program overview, student work, and events pages, considering time and budget constraints.
  • Design a new page, 'New Student,' from scratch for implementation on the site.

Solution

We streamlined the user flow, refreshed the UI kit, updated key pages, and seamlessly integrated a new 'New Student' page.
